# RtMateriais Site

Projeto comercial para apresentacao de uma loja de materiais de construcao, com frontend em React e API em .NET.

## Estrutura

- `src/`: interface do site e paginas do catalogo.
- `public/`: arquivos estaticos do frontend.
- `RtMateriais.Api/`: API em ASP.NET para evolucao do projeto.

## Como rodar o frontend

```bash
npm install
npm start
```

O site abre em `http://localhost:3000`.

## Como gerar build

```bash
npm run build
```

## Como rodar a API

Na pasta `RtMateriais.Api`, execute:

```bash
dotnet run
```
